# Motto Web Tool
A web tool for running [Motto](https://github.com/MichaelMW/motto.git)

## Requirements
- Python 3.7
- Node.js 14.3.0

## Getting started
Install Node packages
```
$ npm install
```
Start up the node webserver
```
$ npm run start
```

Start up the Flask webserver
```
$ python server.py
```

Go to `localhost:3000` in your browser
